# Onegin
## Programm sortes file of strings

### How to use

**Write into shell**
```shell
make
build/onegin
```
**If you want to input your own files enter in shell**
```shell
build/onegin <input_file> <output_file>
```
**Also --help flag is available**
```shell
build/onegin --help
```
or
```shell
build/onegin -h
```
